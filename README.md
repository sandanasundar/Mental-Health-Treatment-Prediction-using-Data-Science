# 🧠 Mental Health Treatment Prediction

A machine learning project that analyzes a real-world mental health survey dataset to identify factors influencing whether individuals seek mental health treatment. The project includes data preprocessing, exploratory data analysis (EDA), visualization, and predictive modeling using multiple machine learning algorithms.

## 📌 Project Overview

Mental health is an important aspect of overall well-being. This project explores survey data to understand patterns related to mental health treatment and builds predictive models to classify whether a person is likely to seek treatment.

## 🎯 Objectives

- Clean and preprocess the dataset
- Perform Exploratory Data Analysis (EDA)
- Visualize important trends and relationships
- Train multiple machine learning models
- Compare model performance
- Identify the most influential features

## 📂 Dataset

**Dataset:** Mental Health Survey Dataset (`survey.csv`)

The dataset contains information such as:
- Age
- Gender
- Country
- Family History
- Remote Work
- Company Benefits
- Work Interference
- Mental Health Treatment (Target Variable)
